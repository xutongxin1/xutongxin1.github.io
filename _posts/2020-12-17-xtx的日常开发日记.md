instanceof 是 Java 的一个二元操作符，类似于 ==，>，< 等操作符。

instanceof 是 Java 的保留关键字。它的作用是测试它左边的对象是否是它右边的类的实例,返回Boolen

如

```java
Object testObject = new ArrayList();
if(testobject instanceof ArrayList) //true
```
